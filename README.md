# 🌍 PIIT Segovia
## PLATAFORMA INTEGRAL DE INTELIGENCIA TERRITORIAL · SIG MUNICIPAL

Plataforma de Inteligencia Territorial para el municipio de Segovia (Antioquia, Colombia) con enfoque de modelo escalable. Integra análisis geoespacial, datos abiertos y visualización avanzada para apoyar la toma de decisiones, la planificación sostenible y el desarrollo de ciudades inteligentes basadas en evidencia.

---

## 🎯 Objetivo
Desarrollar una herramienta digital que permita analizar, visualizar y gestionar información territorial para fortalecer la gobernanza basada en datos.

---

## 🧠 Enfoque
- Inteligencia territorial
- Smart Cities
- Análisis geoespacial (SIG)
- Datos abiertos
- Planeación urbana sostenible

---

## ⚙️ Tecnologías
- HTML, CSS, JavaScript
- Leaflet (SIG)
- GitHub Pages (despliegue web)

---

## 🌎 Alcance
Proyecto piloto en Segovia con potencial de replicabilidad en Colombia y América Latina.
